# Insert your RNA coverage files in this subfolder.

RNA coverage files: 
your RNA coverage files (2 per sample, corresponding to forward and reverse coverage) should be put in the provided RNAbigwig-global subfolder. 
We recommend using the global, min-normalized files (see HRIBO documentation). Expected file extension "min.reverse.global.bw" for reverse and "min.forward.global.bw" for forward coverage.
